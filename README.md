AGL Application
===============

Base structure to create AGL applications. HTML5 ready with [HTML5 Boilerplate](http://html5boilerplate.com/) v4.0.2.

Install [**AGL**](https://github.com/agl-php/agl) once and run an unlimited number of applications.
